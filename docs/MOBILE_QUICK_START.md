# 移动端快速开始指南

5 分钟快速上手 QuantMesh 移动端访问。

## 🚀 最快速度开始

### 步骤 1: 启动系统（30秒）

```bash
cd /path/to/opensqt_market_maker
./quantmesh
```

### 步骤 2: 查看地址（10秒）

系统会显示：
```
✅ Web服务器已启动，可通过 http://0.0.0.0:28888 访问
```

查看你的 IP 地址：
```bash
# macOS
ifconfig | grep "inet " | grep -v 127.0.0.1

# Linux
ip addr show | grep "inet " | grep -v 127.0.0.1
```

### 步骤 3: 手机访问（1分钟）

1. 确保手机和电脑在同一 WiFi
2. 打开手机浏览器（Safari/Chrome）
3. 输入：`http://你的IP:28888`
4. 例如：`http://192.168.1.100:28888`

### 步骤 4: 安装 PWA（2分钟）

**iOS:**
1. 点击底部分享按钮 ⬆️
2. 选择"添加到主屏幕"
3. 点击"添加"

**Android:**
1. 点击右上角菜单 ⋮
2. 选择"安装应用"
3. 确认安装

### 步骤 5: 开始使用

点击主屏幕的 QuantMesh 图标，开始使用！

## ⚠️ 重要安全提示

在开始使用前，请确保：

1. **禁用 API 提现权限**
   - 登录交易所后台
   - 找到 API 管理
   - 禁用"提现"和"转账"权限

2. **启用 IP 白名单**
   - 添加你的服务器 IP
   - 限制只能从特定 IP 访问

3. **使用子账户**
   - 创建专门的子账户
   - 只转入必要的资金

## 🔍 权限检测

系统启动时会自动检测 API 权限：

```
🔐 [binance:BTCUSDT] 开始检测 API 权限...
✅ [binance:BTCUSDT] API 权限检测通过 (安全评分: 80/100, 风险等级: low)
💡 建议：启用 IP 白名单限制以提高安全性
```

如果看到警告：
```
🚨 [binance:BTCUSDT] API 密钥存在安全风险！
⚠️ 危险：API 密钥具有提现权限！强烈建议禁用
```

**立即停止系统，修改 API 权限！**

## 🌐 远程访问（可选）

如果需要在外网访问，推荐使用 Tailscale：

### 安装 Tailscale（5分钟）

1. **服务器端**
   ```bash
   # macOS
   brew install tailscale
   sudo tailscale up
   
   # Linux
   curl -fsSL https://tailscale.com/install.sh | sh
   sudo tailscale up
   ```

2. **手机端**
   - 下载 Tailscale 应用
   - 使用同一账号登录

3. **访问系统**
   - 查看 Tailscale IP（通常是 100.x.x.x）
   - 访问：`http://100.x.x.x:28888`

### 为什么选择 Tailscale？

- ✅ 零配置
- ✅ 端到端加密
- ✅ 无需公网 IP
- ✅ 免费使用
- ✅ 非常安全

## 🆘 常见问题

### 无法访问

**问题：** 手机浏览器显示"无法连接"

**解决：**
1. 确认服务器正在运行
2. 确认手机和电脑在同一 WiFi
3. 检查防火墙设置
4. 尝试使用 IP 地址而非域名

### PWA 无法安装

**问题：** 没有"添加到主屏幕"选项

**解决：**
1. 确保使用 Safari (iOS) 或 Chrome (Android)
2. 刷新页面
3. 清除浏览器缓存
4. 更新浏览器到最新版本

### 系统提示 API 权限危险

**问题：** 启动时显示提现权限警告

**解决：**
1. **立即停止系统**（Ctrl+C）
2. 登录交易所后台
3. 找到 API 管理
4. 禁用"提现"和"转账"权限
5. 重新启动系统

### 图标显示异常

**问题：** PWA 图标不正确

**解决：**
1. 这是正常的（当前使用占位符）
2. 不影响功能使用
3. 生产环境需要生成真实图标
