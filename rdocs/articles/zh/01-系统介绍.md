## 什么是 QuantMesh？

**QuantMesh Market Maker** 是一个高性能、低延迟的加密货币做市商系统，专注于永续合约市场的单向做多无限独立网格交易策略。

### 系统定位

- **类型**: 高频量化交易系统
- **开发语言**: Go 1.21+
- **交易策略**: 网格交易（Grid Trading）
- **目标市场**: 永续合约市场
- **交易方向**: 单向做多（低买高卖）

### 核心特点

#### 1. 毫秒级响应
- ✅ 全 WebSocket 驱动，拒绝轮询延迟
- ✅ 实时价格和订单流，确保决策及时性
- ✅ 基于事件驱动的架构设计

#### 2. 多交易所支持
- ✅ 支持 20+ 主流交易所
- ✅ 统一的接口抽象，易于扩展
- ✅ 自动适配各交易所的 API 差异

#### 3. 智能网格策略
- ✅ **固定金额模式**: 资金利用率更可控
- ✅ **超级槽位系统**: 智能管理挂单与持仓状态，防止并发冲突
- ✅ **无限网格**: 不设上下限，自动跟随价格波动

#### 4. 强大的风控系统
- ✅ **主动风控**: 实时监控 K 线成交量异常，自动暂停交易
- ✅ **资金安全**: 启动前自动检查余额、杠杆倍数与最大持仓风险
- ✅ **自动对账**: 定期同步本地与交易所状态，确保数据一致性
- ✅ **订单清理**: 自动清理过期订单，防止订单堆积

#### 5. 高并发架构
- ✅ 基于 Goroutine + Channel + Sync.Map 的高效并发模型
- ✅ 细粒度锁设计，避免死锁
- ✅ 原子操作保证数据一致性

## 实战验证

经过数个版本迭代，我们已经使用此系统交易超过 **1 亿美元** 的虚拟货币。

### 实际案例

**交易 ETHUSDT，0 手续费，价格间隔 1 美元，每笔购买 300 美元：**

- 每天的交易量将达到 **300 万美元以上**
- 一个月可以交易 **5000 万美元以上**
- 只要市场是震荡或向上将持续产生盈利

**风险控制示例：**

- 3 万美元保证金可以保证下跌 1000 个点不爆仓
- 通过不断交易拉低成本，只要回涨 50% 即可保本
- 如果出现单边极速下跌，主动风控系统将会自动识别立刻停止交易
- 当市场恢复后才允许继续下单，不担心插针爆仓

**盈利示例：**

ETH 从 3000 点开始交易：
- 价格下跌到 2700 点，亏损约 3000 美元
- 价格涨回 2850 点以上已经保本
- 价格涨回 3000 点，盈利在 1000-3000 美元

## 快速开始

### 环境要求

- Go 1.21 或更高版本
- 网络环境需能访问交易所 API
- 交易所账户和 API 密钥

### 安装步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/dennisyang1986/quantmesh_market_maker.git
   cd quantmesh_market_maker
   ```

2. **安装依赖**
   ```bash
   go mod download
   ```

3. **配置系统**
   ```bash
   cp config.example.yaml config.yaml
   # 编辑 config.yaml，填入你的 API Key 和策略参数
   ```

4. **运行系统**
   ```bash
   go run main.go
   ```

更多详细信息，请参阅 [快速入门指南](./02-快速入门.md)。
